Unity 기초

1. UI 시스템

Canvas: UI 요소들을 담는 컨테이너

Button: 클릭 가능한 버튼

Text: 문자 표시

Image: 이미지 표시

Slider: 값 조절 바

2. 오디오

AudioSource + AudioClip 조합

Play(): 소리 재생

Stop(): 소리 정지

volume: 볼륨 조절

loop: 반복 재생

3. 카메라

Camera.main으로 메인 카메라 접근

ScreenToWorldPoint: 화면 좌표 변환

fieldOfView: 시야각 조절

orthographic: 직교/원근 투영

4. 라이팅

Directional Light: 태양광 같은 평행광

Point Light: 전구 같은 점광원

Spot Light: 손전등 같은 원뿔광

Intensity로 밝기 조절

5. 머티리얼

오브젝트의 외관을 결정

Albedo: 기본 색상/텍스처

Metallic: 금속성 정도

Smoothness: 표면 매끄러움

6. 파티클 시스템

불, 연기, 폭발 등 효과 생성

Emission: 방출 설정

Shape: 방출 모양

Color over Lifetime: 시간별 색변화

7. 물리 조인트

Hinge Joint: 경첩 연결

Fixed Joint: 고정 연결

Spring Joint: 스프링 연결
