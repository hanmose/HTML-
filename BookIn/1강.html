Unity 기초

1. Unity 인터페이스

Scene View: 게임 세계 편집 공간

Game View: 게임 실행 화면

Hierarchy: 오브젝트 목록

Project: 파일 관리

Inspector: 오브젝트 속성

2. 게임 오브젝트

모든 것은 GameObject + Component 구조

Transform: 위치, 회전, 크기

Renderer: 화면 출력

Collider: 충돌 감지

Rigidbody: 물리 효과

3. C# 스크립팅

MonoBehaviour 상속하여 스크립트 작성

Start(): 시작 시 한번 실행

Update(): 매 프레임 실행

FixedUpdate(): 물리 연산용

4. 입력 처리

Input.GetKey(KeyCode.W) - 키보드

Input.GetMouseButtonDown(0) - 마우스

5. 프리팹

재사용 가능한 오브젝트 템플릿

Project 창으로 드래그하여 생성

6. 좌표계

X: 좌우, Y: 상하, Z: 앞뒤

Transform.position으로 위치 제어

7. 애니메이션

Animation 창에서 키프레임 설정

Animator Controller로 상태 관리

8. 태그와 레이어

GameObject.FindWithTag() 활용

충돌 및 렌더링 제어
